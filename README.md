# VIBE Portfolio Site

A minimal, bold Astro + Tailwind portfolio for NANDIKA SOI with placeholder content, newsletter signup, and Cloudflare Pages deployment readiness.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local dev server:
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```

## Deploy to Cloudflare Pages

1. Create a new Cloudflare Pages project.
2. Connect this repository.
3. Set the build command to:
   ```bash
   npm run build
   ```
4. Set the output directory to:
   ```bash
   dist
   ```
5. Deploy the project.

## Notes

- Replace placeholder content in the pages and components as needed.
- The newsletter form is a frontend placeholder and does not submit anywhere yet.
